# Serverless-CRUD-API-using-AWS
This project is a serverless REST API built using AWS Lambda, API Gateway, and DynamoDB. It performs CRUD operations and demonstrates scalable backend architecture without managing servers.
Designed and deployed a fully serverless RESTful API enabling CRUD operations with AWS Lambda and API Gateway.
Leveraged DynamoDB for high-performance, low-latency data storage and retrieval.
Improved system efficiency by optimizing Lambda execution and minimizing response time.
Implemented centralized logging and monitoring using CloudWatch to enhance observability and maintain system stability.
Tech: AWS Lambda, API Gateway, DynamoDB, Cloud Watch, PostMan.
